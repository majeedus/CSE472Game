CSE 471 - Project 1 - Piano

Sample Xml Configuration

    <?xml version="1.0" encoding="utf-8"?>
    <score bpm="120" beatspermeasure="4">
    <instrument instrument="Piano">
    <note measure="2" beat="1" duration="0.2" note="CompletePiano\pedald.wav" amplitude="0.2" keypressure="1.2"/>
    <note measure="2" beat="1" duration="1.9" note="CompletePiano\G4l.wav" setdampening="1.0" undampen="1.3"/>
    <note measure="2" beat="2.5" duration="0.2" note="CompletePiano\pedalu.wav" amplitude="0.2"/>
    <note measure="2" beat="3" duration="0.9" note="CompletePiano\E4l.wav" attack="0.05" decay = "0.05" release="0.05"/>
    <note measure="2" beat="4" duration="0.9" note="CompletePiano\A4l.wav" speed=".9"/>
    </instrument>
    </score>

Grading Elements

  • 10 - Polyphony : A map is used to keep track of the different wavetables from the loaded wave files to combine notes. This allowed the factory to keep track of the various notes for each instrument.
  • 20 - Envelope Generation : A function that allows attack, decay, sustain and release times, during which the amplitude is changed.
  • 30 - Pedal simulation : The pedal is simulated using a decaying exponential for the dampening as well as a value for time that can be set to stop the dampening at a certain time during the note playback.
  • 35 - Basic Dynamics : For a quick keypress, a value that lowers the attack duration if the key is pressed quickly.
  • 40 - Pedal noise : the pedal noise is simulated by the short sound of a pedal being pressed and released.
  • 50 - Advanced Dynamics : Pressure can be simulated by adjusting the amplitude, attack and delay to stretch the initial effect if the pressure is low and condense if the pressure is high.
  • The piano score is the song "The Old Man is Snoring"
